Resume materi Javascript Refreshment

Adapun 3 poin utama yang didapatkan dari pembelajaran materi Javascript Refreshment adalah sebagai berikut:

1. Pengertian Javascript dan variabel pada Javascript
Javascript merupakan bahasa pemrograman tingkat tinggi, atau dapat disebut juga High-Level, Scripting, Untyped dan interpreted. Dikatakan bahasa pemrograman tingkat tinggi, karena Javascript merupakan bahasa pemrograman yang kodenya sudah sangat menyerupai bahasa manusia, yaitu sudah mirip dengan bahasa inggris yang sering kita gunakan. Javascript dibuat sebagai penetrasi pada web browser, yang dimana hal tersebut berkaitan dengan seluruh web browser apapun. Javascript sendiri memiliki beberapa library yang populer digunakan oleh para developer, antara lain adalah vue, react, dan lain-lain. Pada Javascript terdapat declaration yang berperan untuk memproses pembuatan variabel untuk menyimpan data. Untuk membuat variabel pada Javascript sendiri terbagi menjadi 3 yaitu var, let, dan const. Var memiliki sistem functional scope, yang artinya VAR tidak dapat diakses jika kita mencoba mengaksesnya dari luar function, tetapi masih bisa diakses dari luar block lain seperti perulangan atau percabangan. Sedangkan Let dan Const tidak dapat diakses baik di luar function, percabangan, maupun perulangan. Data pada Var dan Let dapat diubah-ubah, dan memberikan output sesuai dengan data terbaru, sedangkan data pada Const merupakan data yang tidak dapat diubah-ubah.

2. Tipe data primitif pada Javascript, Object, Destructuring, Spread Syntax
Dalam pemrograman primitif merupakan unit pemrosesan terkecil dan merupakan elemen yang paling sederhana yang tersedia dalam bahasa pemrograman. Beberapa tipe data primitif pada Javascript antara lain adalah string, boolean, number, bigint, undefined, null, dan symbol. Sedangkan objects adalah unit yang menyimpan properti dan fungsi (method). Objects pada Javascript sendiri antara lain adalah Array, Function, Date, Set, Map, Weak Set, dan Weak Map. Destructuring merupakan ekspresi dari Javascript yang memungkinkan untuk menyalin nilai dari array atau properti dari objek ke dalam variabel yang berbeda. Spread syntax dapat digunakan ketika semua elemen dari object atau array perlu dimasukkan ke dalam semacam daftar. 

3. Method pada Javascript, Control Flow pada Javascript, Function pada Javascript, Document Object Model (DOM) pada Javascript
Method merupakan sebuah fungsi yang terkait dengan object, dalam membuat programnya menjadi se-sederhana mungkin sesuai dengan kegunaannya masing-masing. Beberapa method yang terdapat pada Javascript antara lain adalah Concat (Menggabungkan dua atau lebih array dan mengembalikan salinan array yang digabungkan), Map (Membuat array baru dengan hasil memanggil fungsi untuk setiap elemen array), Foreach (Memanggil fungsi untuk setiap elemen array), Slice (Memilih bagian dari array dan mengembalikan array baru), Filter (Membuat array baru dengan setiap elemen dalam array yang lulus seleksi), dan Reduce (Melakukan operasi pada setiap elemen array menjadi nilai tunggal dari kiri ke kanan). Control Flow merupakan alur program jika suatu kondisi terjadi sesuai dengan keinginan kita. Control flow sendiri terbagi menjadi 2 yaitu Pengulangan (Loop/Iterasi) antara lain For, While, Do While. Dan yang kedua adalah pengkondisian (Percabangan) antara lain if else, switch, block, try catch, break, continue, dan throw. Function pada Javascript merupakan sebuah objek, karena memiliki properti dan juga method. Function digunakan untuk melakukan serangkaian komputasi atau prosedur yang dapat digunakan secara terus menerus. Document Object Model (DOM) dapat didefinisikan sebagai API untuk HTML yang merepresentasikan halaman web pada suatu dokumen menjadi sebuah object. DOM sendiri terbagi menjadi 3 yaitu Selection Method, Manipulation, dan Event. 
- Pada DOM Selection Method terdapat beberapa event seperti getElementById() yang berfungsi sebagai elemen, kemudian terdapat getElementsByTagName() dan getElementsByClassName() yang berfungsi sebagai HTMLCollection, kemudian terdapat querySelector() yang berfungsi sebagai element, dan terdapat querySelectorAll() yang berfungsi sebagai nodeList.
- Pada DOM Manipulation terdapat beberapa event seperti element.innerHTML (digunakan untuk merubah isi dari tag yang sudah diseleksi), kemudian terdapat element.style.<propertyCSS> (digunakan merubah style dari tag yang sudah diseleksi), dan terdapat element.setAttribute() (digunakan untuk mengelola atau memanipulasi attribute yang sudah diseleksi), dan terdapat element.classList.add() (digunakan untuk mengelola atau memanipulasi class yang sudah diseleksi).
- Pada DOM event terdapat beberapa event dengan berbagai macam kegunaannya, seperti onclick (digunakan ketika pengguna mengklik mouse), kemudian terdapat onchange (digunakan dalam kombinasi dengan validasi bidang input), onblur (digunakan ketika meninggalkan kolom input), onmouseover (digunakan ketika pengguna mengarahkan mouse ke atas pada suatu objek), kemudian terdapat onmouseout (digunakan ketika pengguna mengarahkan mouse ke luar pada suatu objek), dan terdapat oncopy (digunakan ketika pengguna mengcopy suatu objek).

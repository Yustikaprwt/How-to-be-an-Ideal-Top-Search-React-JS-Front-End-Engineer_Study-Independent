Berapa banyak kekurangan dalam penulisan kode tersebut?
Jawaban: 2 kesalahan

Bagian mana saja terjadi kekurangan tersebut?
Jawaban: 
- Pada penamaan deklarasi class user dan userservice
- Pada penamaan fungsi getalluser dan getuserbyid

Tulskan alasan dari tiap kekurangan tersebut?
Jawaban: 
- Seharusnya pada penamaan deklarasi class ditulis dengan format Uppercase.
- Seharusnya pada penamaan fungsi ditulis dengan format penulisan camelCase.
